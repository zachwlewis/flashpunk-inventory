FlashPunk Inventory
===================

A demonstration of using [FlashPunk](http://useflashpunk.net) to create a contained `Inventory` object.

Requirements
------------

Built with [FlashPunk 1.7.2](https://github.com/Draknek/FlashPunk/releases/tag/v1.7.2).
